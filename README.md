# Описание алгоритма программы

1. На вход приходит массив *array*.
2. Создаётся новый массив *newArray*.
3. Создаётся переменная i, равная 0.
4. Создаётся цикл, который раотает, пока i меньше длины массива *array*.
    * Проверяется длина i-го элемента массива *array*
        * Если длина меньше или равна 3, то этот элемент добавляется в конец массива *newArray*.
        * Если длина больше 3, ничего не происходит.
    * i увеличивается на 1.
5. Программа возвращает заполненный массив *newArray*, выводя его в консоль.
